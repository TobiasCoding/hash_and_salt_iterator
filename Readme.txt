It's a simply script you'll be able to get the Hash of a file gived as input and then iterate the hash with salt (ramdom string of letters and numbers).
The program create a file to serve as a route to replicate the process. Therefore, it allow validate the original file thanks to the file which behaves as a key.
The propose of this program is give a strong security of the hash to determinated uses, as publish in public the hash code like a commercial secret.
